    API (same as server routes)

    • GET /api/books — list all books
    • GET /api/books/:id — one book by id
    • POST /api/books — create (body: id, title, author, year, genre, summary, price)
    • PUT /api/books/:id — update (body must not include id)
    • DELETE /api/books/:id — remove book (empty 204 on success)
